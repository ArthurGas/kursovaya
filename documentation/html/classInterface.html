Класс интерфейса взаимодействия с пользователем

Конструтор обнуляет все значения. Интрефейс реализован для командной строки с помощью библиотеки boost_program_options

bool Interface::set_options ( int  argscount,
char *  argvectors[],
Error &  er 
)

Метод установки опций интерфейса

Реализация с помощью boost_program_options. Поддержка опций: "порт", "файл базы пользователей" и "файл для записи ошибок". Метод значений не возвращает

Аргументы
[in]argscountколичество агрументов командной строки, принимается из параметров функции main
[in]argvectorsуказатель на массив со значениями агрументов командной строки, также принимается из параметров функции main
Возвращает
False или true, означающее, нужно ля запускать программу(для вызова справки)
Предупреждения
Использование несуществующих опций может привести к аварийному останову
Аргументы
[in]erобъект класса исключений для установки параметра критичности
