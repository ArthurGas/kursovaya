Calculator Server  1.0
Сервер для вычисления произведения вектора типа float. Вычисления проводятся только только для авторизованных клиентов, перед вычислениями производтится авторизация
 
  • base_cont : Auth
  • base_file : Interface
  • client_addr : Connection
  • count : Calculation
  • iscritical : Error
  • log_file : Interface
  • port : Interface
  • queue_len : Connection
  • SALT : Auth
  • serv_addr : Connection
  • size : Calculation
  • sock : Connection
  • work_sock : Auth , Calculation
